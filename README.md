# My-LMS图书管理系统

#### 介绍
一个简单的图书管理系统

#### 软件架构
软件架构说明
前端：vue3+element-plus
后端：springboot+mybatis
数据库：mysql

#### 安装教程

1.  解压文件
2.  配置yml文件数据库连接信息
3.  执行sql脚本
